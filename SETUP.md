# ===== Настройка Telegram бота =====
# Создайте бота через @BotFather в Telegram:
# 1. Откройте Telegram, найдите @BotFather
# 2. Отправьте /newbot
# 3. Следуйте инструкциям
# 4. Скопируйте токен бота
# 5. Узнайте свой chat_id через @userinfobot

# Для Cloudflare Workers:
# 1. Создайте аккаунт на https://dash.cloudflare.com
# 2. Перейдите в Workers & Pages
# 3. Создайте новый Worker
# 4. Вставьте код из telegram-worker.js
# 5. Замените BOT_TOKEN и CHAT_ID
# 6. Deploy
# 7. Скопируйте URL Worker

# Для Email через FormSubmit:
# 1. Просто замените YOUR_EMAIL в index.html на ваш email
# 2. Первый раз подтвердите email через письмо от FormSubmit
