# Сайт-визитка студии Николаева Александра

Сайт-визитка в стиле **Textual** для студии разработки программного обеспечения.

## 🚀 Быстрый старт

Подробные инструкции:
- ⚡ [QUICKSTART.md](QUICKSTART.md) — быстрый старт (5 минут)
- 📖 [DEPLOY.md](DEPLOY.md) — полная инструкция по настройке
- 📝 [SETUP.md](SETUP.md) — настройка Telegram бота

## 🎨 Стиль Textual

Сайт выполнен в минималистичном терминальном стиле:
- Моноширинный шрифт JetBrains Mono
- Тёмная цветовая схема
- Элементы интерфейса терминала (точки, заголовок)
- Мигающий курсор
- Кодовые префиксы (`>`, `$`, `#`)
- Scanlines эффект

## 📧 Получение заявок

### На Email (бесплатно):
Используется **FormSubmit.co** — просто замените email в `script.js`

### В Telegram (бесплатно):
Используется **Cloudflare Workers** + Telegram Bot API

## 🚀 Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Запуште код
3. Включите GitHub Pages → GitHub Actions
4. Сайт доступен по адресу: `https://USERNAME.github.io/REPO/`

Автоматический деплой при каждом push в main/master.
